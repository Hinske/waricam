# Text-Tool Setup

## 2 Dateien manuell herunterladen:

### 1. opentype.min.js → js/opentype.min.js
Download: https://cdn.jsdelivr.net/npm/opentype.js@1.3.4/dist/opentype.min.js
Speichern als: `js/opentype.min.js`

### 2. Standard-Font → fonts/default.ttf
Empfohlene Fonts (einen davon als `default.ttf` speichern):
- **DIN 1451 Mittelschrift** — Industrie-Standard, gut lesbar
- **Open Sans** — https://fonts.google.com/specimen/Open+Sans
- **Arial** — von Windows kopieren: `C:\Windows\Fonts\arial.ttf`

Alternativ akzeptiert das Tool auch: `fonts/arial.ttf`, `fonts/DIN1451Mittelschrift.ttf`, `fonts/OpenSans-Regular.ttf`

## Benutzung
- **TX** in CommandLine oder Ribbon-Button "Text"
- Text eingeben → Höhe eingeben (Default: 20mm) → Platzierungspunkt klicken
- **FONT** tippen um andere .ttf/.otf Datei zu laden
